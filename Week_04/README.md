### 深度优先DFS 和 广度优先BFS ###
    共同特点：
        每个结点都要访问一次且仅访问一次
        
    DFS: deep first search
    BFS: breadth first search

### 贪心算法 ###
    Greedy 一般用来解决最优化的问题，如：最小生成树、哈夫曼编码等
    一般也用作辅助算法 或者对于结果要求不太精确的问题
    
    贪心算法 与 动态规划 的不同点：对每个子问题的解决方案都做出选择，不能回退

### 二分查找 ###
    前提：
    1、单调递增或者递减
    2、存在上下界
    3、能够通过索引访问
